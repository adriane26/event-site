Creating New Events

To create a new event, click on the "Events" tab in the admin menu, then clik on the orange "Add New" button. This will bring you to the New Event form where you can create new events. The minimum needed to create an event are the "Event Name", "Event URL", "Event Location", and some content in the "About page Content". These are the only required fields.

The different fields in the form create the different parts of an event e.g. name, location, url etc. The fields work as follows:

Field Name
What it saves in the database
Is this field required?
Event Name The name of the event. Yes
Event URL The path of the URL you want the event to have. E.g. if you enter "seattle2016" the URL for your event will be "https://interopevents.com/seattle2016" Yes
Display Language The language that will be used in the lang attribute for the event. The default is English No
Event Registration Link This is the link to register for the event. It must be a fully formed URL including "http://" or "https://". If you enter a value here there will be a register button added to your event that follows this link. No
Technical Topics Once they are over events are automatically added to the past events page. These are what will be listed as the technical topics in the past events table. No
Event Location Where the event takes place. Yes
Choose Event Continent Assigns the color that will be applied to the events header menu for this event No
Event Dates (Must include time of day) The start and end dates for the event. They must include the time of day No
Display on the colored events menu underneath gray "Home" menu? Determines whether the event will appear in the colored menu in the header or not. The default is false No
About page Content The html content that will appear in the "About" page for the event. This will always be the first tab for the event. Yes
Venue Name This is the name of the venue. If you add a "Venue Name" it will create a tab for you called "Venue" No
Address The address that will show for the venue No
Parking Information Text that will show on the Venue tab No
Select Venue Image Image to show on Venue tab No
Select Header Image Image to show on masthead for event No
Select Speakers Opens a modal window where you can select speakers by dragging from the left side to the right. All speakers on the right will be displayed in the order shown No
Cancel Cancels thew new event N/A
Preview Pops up modal window with preview of event N/A
Save as Draft Saves the event as a draft, but does not publish N/A
Publish Now Saves the event and publishes it immediatly to the dom N/A
